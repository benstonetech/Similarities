Similarities is a Web App that allows users to compare two files for similarities in three different ways

  1. Lines
  2. Sentences
  3. Substrings

Each method is useful in their own way, for example, lines could be used for comparing source code files whereas sentences is better suited to comparing longer extracts of text

I have prepared some example documents so you can quickly see how the application is able to find and higlight similarities between files.
